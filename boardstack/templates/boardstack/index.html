boardstack — stack-of-CAs experiment

Experimental app. Builds a stack of N class-4 CA boards (each a cell8 LUT upcasted from a mandelhunt-derived 7→1 rule), runs them with variable per-board rates and gene-encoded inter-board wiring, and asks whether the resulting stack can transform an input byte into a target byte for arbitrary (p, q) test pairs without per-pair training.

Isolated from caformer's production paths — no QRPair writes, no chat dispatch interception. If it underperforms we delete the app without fallout.
