# English content words for Lingua flashcard decks.
#
# Hand-curated core vocabulary (~200 items) biased toward practical
# nouns/verbs/adjectives that round-trip cleanly through machine
# translation. Pure function words ("the", "of", "a") are omitted —
# MT models translate them inconsistently in isolation and the cards
# would be noise. Order is rough frequency; the build command slices
# from the top.
#
# One lemma per non-comment line. `lingua_build_deck --size N` takes
# the first N non-comment lines and routes each through the translator
# pipeline to produce target-language cards.
