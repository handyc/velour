mangelbrot

Generates an 8 × 8 sheet of 64 × 64 Mandelbrot sprites — each a random view through a 256-entry palette built from 4 RGB base colours. The bases drift across the sheet so neighbouring sprites feel related. The most populated slot in each sprite is rendered as magenta (#ff00ff) so a downstream tool can colour-key it as transparent.

Each sprite picks a curated Mandelbrot center, perturbs it, and chooses a random log-span in 2-10‥20. The 256-entry palette is built as four 64-step ramps through the 4 RGB bases. Per-sprite a tiny ±6/channel drift is applied to each base, so the sheet's colour scheme walks smoothly instead of jumping.
