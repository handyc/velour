Det — boundaries of determinism

A deterministic engine has just one job: same inputs, same answer. But on a grid of cells following a uniform local rule, identical inputs can still yield four wildly different worlds. Stephen Wolfram's classification names them: Class 1 evolves to a single colour; Class 2 settles into small stable or periodic patches; Class 3 boils into statistical noise; Class 4 — the edge of chaos — produces localized structures that drift, collide, and compute. Rule 110 is the canonical 1-D Class 4 rule; Matthew Cook proved it Turing-complete in 2004.

Det searches for Class 4 analogs on Automaton's hexagonal substrate (cell + 6 neighbors, n ∈ {2, 3, 4} colors). It generates random 7-tuple rulesets, steps each forward, measures activity, block entropy, cycle length, and color diversity, and ranks candidates by how Rule-110-like their dynamics look. Promising survivors can be promoted to Automaton for interactive play.
