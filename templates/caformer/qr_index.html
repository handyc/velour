Q→R curated training · evolve the model to produce specific responses byte-for-byte

Each row is a (prompt, expected) pair the trainer will evolve toward. The CLI runs long bursts of GA + polish + random restarts (the inline pipeline-autotrain runs ~30s; this is hours). Every improvement updates the row in place, so you can deploy the current best as a TrainedModel and chat with it at any time — even mid-training.

# Train a single pair for 1 hour:
venv/bin/python manage.py caformer_qr_train --pair-id <PK> --hours 1
# Train every pair where best_exact=False, round-robin:
venv/bin/python manage.py caformer_qr_train --all-pending --hours 8
# Create-and-train in one shot:
venv/bin/python manage.py caformer_qr_train --prompt "hi" --expected "hello" --hours 2 --pop 32
