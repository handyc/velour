Doom CA

A Doom-shaped game whose world is a 4-colour hex CA running under a sealed spoeqi pact's rule. Cell states map to ground / wall by threshold; player and monsters are app-tracked overlays. The CA's class-4 dynamics make walls shift slowly past as the camera follows the player.

Determinism is inherited: two players with the same pact + same component + same keypress sequence get a byte-identical game. In fleet mode you can pick any of the 64 components — each is a different rule = a different "level".
