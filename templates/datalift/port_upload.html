Port a Legacy App

Drop a mysqldump here. Datalift generates a Django models.py, admin.py, and a starter ingest/table_map.json — the same output as manage.py port, returned as a zip you unpack into your project.

Everything runs locally in this Django process. The dump is read, parsed, and discarded — no copies are persisted, nothing is sent anywhere. Only the assembled zip is streamed back to the browser.

What the zip contains
  • <app>/models.py — one class per legacy table with inferred field types (EmailField / URLField / SlugField / TextChoices for ENUMs / FK resolution / Laravel conventions).
  • <app>/admin.py — one ModelAdmin per model with inferred list_display, search_fields, list_filter, date_hierarchy, raw_id_fields.
  • <app>/ingest/table_map.json — starter map for manage.py ingestdump — value_maps for ENUMs, drop_columns for Laravel cruft, synthesize rules for users without usernames, Laravel bcrypt rewrite when a password column is found.
  • README.md — the step-by-step to migrate and ingest.
