Anonymize a Database File

Upload any SQLite database file. Datalift will anonymize all non-primary-key data and return the anonymized version for download.

Strings are replaced character-by-character (uppercase→uppercase, lowercase→lowercase, digit→digit). Numbers preserve magnitude. Dates shift randomly. Primary keys are preserved for referential integrity.
