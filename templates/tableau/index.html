Tableau

A Tarski's World clone — arrange cubes, tetrahedra, and dodecahedra on a board, write first-order sentences about them, and watch the truth values update live as the world changes. Square boards take the classic LPL predicate set (LeftOf, RightOf, FrontOf, BackOf, SameRow, SameCol, Adjoins, Between). Hex boards use a native set (Adjacent, NorthOf, SouthOf, SameQAxis / SameRAxis / SameSAxis, BetweenHex). Shared predicates (Cube, Tet, Dodec, Small, Medium, Large, Larger, Smaller, SameSize, SameShape) work on both.
