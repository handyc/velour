Algorithm parity

PRNG (mulberry32), genome layout (N=2^14 prototypes, parallel keys+outs Uint8Arrays), buildBins, nearest-prototype lookup with squared-Euclidean distance — all imported directly from hexnn_engine.mjs. The same module the browser bench uses. The C distillation (condenser.distill_hexnn.distill_hexnn_esp32s3(with_tft=True)) uses the same algorithm written out in C99; this emulator and the device should agree at the same seed (modulo timing jitter from real-world drift in esp_random()).
