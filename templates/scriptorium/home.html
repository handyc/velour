Configuration

Project paths live in environment variables and the Velour database, not in this repo. Set them per-host: 
export DANWSI_LOCAL_PATH=/path/to/danwsi
export DANWSI_DB_PATH=$DANWSI_LOCAL_PATH/db.sqlite3
# Optional remote / staging:
export DANWSI_REMOTE_HOST=...
export DANWSI_REMOTE_USER=...
export DANWSI_REMOTE_PATH=...
export DANWSI_REMOTE_PYTHON=...
export DANWSI_SSH_KEY_PATH=$DANWSI_LOCAL_PATH/deploy/.ssh/deploy_key
