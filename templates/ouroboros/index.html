The metachain in one sentence

Take a K=4 hex CA rule — a lookup table of 47 = 16,384 entries. Reshape it as a 128 × 128 grid (since 16,384 = 128²). Run the rule on that grid for 16 ticks. The output is another 16,384 cells, hence another rule. Iterate. A *class-4 quine chain* is one where every iterated rule stays at the edge of chaos. A *class-4 fixed point* is a rule that maps to itself under this operation AND is itself class-4. Find one and the chain has infinite class-4 depth — the structure we hunt for.
