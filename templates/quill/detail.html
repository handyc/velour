Compose key cheat sheet

Fast path (recommended): hold Alt and tap a diacritic mark, release Alt, then tap the base letter — two keystrokes per diacritic.

Examples: Alt+. s → ṣ · Alt+= a → ā · Alt+' s → ś · Alt+, c → ç · Alt+v s → š. Marks that need Shift (~ ^ : * ( ") need Alt+Shift, e.g. Alt+Shift+~ n → ñ.

Slow path (no Alt needed): type \ (a single backslash) then a diacritic mark then a base letter, e.g. \.s → ṣ. Type \\ to insert a literal backslash. Ctrl+; / Cmd+; also opens compose mode. Esc cancels; Tab inserts a tab character.
