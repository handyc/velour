Compose key cheat sheet — macOS US Extended replica

Quill's dead-key map mirrors macOS US Extended / ABC Extended exactly. Fast path: hold Alt, tap a prefix letter, release, then tap the base letter.

Examples: Alt+a a → ā · Alt+x s → ṣ · Alt+e s → ś · Alt+n n → ñ · Alt+v s → š · Alt+u u → ü.

Slow path: type \ then prefix then base, e.g. \a a → ā, \x s → ṣ. Type \\ to insert a literal backslash. Esc cancels; Tab inserts a tab character.

Prefix Diacritic Examples
Alt+amacron aa → ā · ae → ē · ai → ī · ao → ō · ay → ȳ · al → ḹ · ar → ṝ
Alt+eacute ea → á · es → ś · en → ń · ec → ć · ez → ź · er → ŕ · em → ḿ
Alt+`grave `a → à · `e → è · `i → ì · `o → ò · `u → ù · `n → ǹ
Alt+6circumflex 6a → â · 6e → ê · 6i → î · 6o → ô · 6u → û · 6s → ŝ · 6h → ĥ
Alt+ntilde nn → ñ · na → ã · no → õ · nu → ũ · ne → ẽ · ny → ỹ
Alt+udiaeresis (umlaut) ua → ä · ue → ë · ui → ï · uo → ö · uu → ü · uy → ÿ
Alt+wdot above wm → ṁ · wn → ṅ · ws → ṡ · wg → ġ · wz → ż · we → ė · wc → ċ
Alt+xdot below (underdot) xs → ṣ · xt → ṭ · xn → ṇ · xm → ṃ · xh → ḥ · xr → ṛ · xl → ḷ · xd → ḍ
Alt+hmacron below hb → ḇ · hd → ḏ · hk → ḵ · hl → ḻ · hn → ṉ · hr → ṟ · ht → ṯ · hz → ẕ
Alt+vcaron (háček) vs → š · vc → č · vz → ž · vn → ň · vr → ř · vd → ď · vt → ť · ve → ě · vh → ȟ
Alt+bbreve ba → ă · be → ĕ · bg → ğ · bo → ŏ · bu → ŭ · bi → ĭ · bh → ḫ
Alt+ccedilla cc → ç · cs → ş · ct → ţ · cn → ņ · cl → ļ · ck → ķ · cr → ŗ
Alt+lstroke lo → ø · ll → ł · ld → đ · lh → ħ · lt → ŧ · li → ɨ · lu → ʉ · lz → ƶ
Alt+kring above ka → å · ku → ů · ky → ẙ · kw → ẘ
Alt+mogonek ma → ą · me → ę · mi → į · mo → ǫ · mu → ų
Alt+zhook above za → ả · ze → ẻ · zi → ỉ · zo → ỏ · zu → ủ · zy → ỷ
Alt+ihorn (Vietnamese) io → ơ · iu → ư
Alt+pcomma below ps → ș · pt → ț
Alt+jdouble acute jo → ő · ju → ű

Capitals work too (Alt+a A → Ā, Alt+x S → Ṣ). Unrecognised combinations insert the literal prefix + base letter so no keystrokes are silently lost.
