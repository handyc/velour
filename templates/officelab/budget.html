Adding the next feature

Past growth has run roughly:

  • each new app costs ~1–3 KB of code (mines was 1393 B, calc 591 B, find 1375 B)
  • the menu engine alone was ~1.6 KB when first added in office4
  • cross-app clipboard infrastructure was small (a few hundred bytes) but the BSS is ~4 KB
  • the LLM ask app was the most expensive single addition at 5.4 KB (curl-execve plumbing + JSON build/parse + chat scrollback)
  • the garden GA breeder was 3.3 KB (mostly thumbnail rendering)

Headroom of {{ budget_left|intcomma }} B means you can comfortably add another small app or two. A second LLM feature would not fit without compression. Per-app slimming would free another 2–5 KB if you really need it (mail can lose To/Subject; sheet's expression parser is the biggest lever inside a single app at ~3.4 KB).
