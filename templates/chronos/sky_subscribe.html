Subscribe to Sky Alerts

A live iCalendar feed you can subscribe to from any calendar app. Refreshes every hour as the underlying forecasts and TLEs change, so the alerts stay accurate.

Currently in the feed:

  • 🌍 Astronomical events next 365 days — eclipses, equinoxes, solstices, planetary conjunctions, and major meteor shower peaks. Alarm at -24 h.
Moon phases are excluded from the feed (too noisy — one event every 7 days, four per month). They're still in the chronos calendar.

Filter the feed

By default the feed includes every category above. To narrow it, append a query string:

  • ?include=passes,transits — only the categories you list. Comma-separated. Allowed: passes, transits, neos, astro.
  • ?exclude=neos — everything except the categories you list. Useful as "default minus astronomy" if you only want the satellite-related stuff.

Example — passes only, no transits or NEOs:
{{ feed_url }}?include=passes

Apple Calendar (Mac, iPhone, iPad)

  1. Tap the webcal:// URL above on the device — it will offer to subscribe directly.
  2. On Mac: File → New Calendar Subscription… and paste the standard URL. Set "Auto-refresh" to Every hour.

Google Calendar

  1. Open calendar.google.com.
  2. Click + next to "Other calendars" → "From URL".
  3. Paste the standard URL above. Click "Add calendar".

Google syncs subscribed calendars roughly every 8-24 hours, so a same-day update may not appear until the next sync.

Outlook / Office 365

  1. Open Outlook on the web → Calendar → Add calendar → Subscribe from web.
  2. Paste the standard URL.

Privacy note

The feed embeds your observer location ({{ feed_url|cut:"http://"|cut:"https://"|truncatechars:0 }}observer's lat/lon) in LOCATION fields and is publicly readable on any host that can reach this server. Don't expose the URL on a public-internet Velour install without a token if your location is sensitive.
