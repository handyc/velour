Conduit — pipelines & routing

Conduit is Velour's dispatcher. A Job describes a unit of work (shell command, Slurm sbatch script, HTTP request, agent task, sensor read, firmware flash). A JobTarget is a registered backend (this host, a VPS, the ALICE HPC cluster, a Pi, an ESP, an ATtiny, a code agent, a generic HTTP endpoint). Submit a job; Conduit routes it to the most appropriate target and executes it — or, for ALICE-bound work, hands you a ready-to- copy sbatch script and waits while you submit it manually.

Phase 1 ships two real executors: local shell and Slurm manual- submit. Other target kinds are stubs until the hardware/agents are wired.
