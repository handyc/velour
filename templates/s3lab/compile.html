What xcc700 supports

if/else, while, function defs & calls, int, char, pointers, arrays, basic arithmetic + bitwise.

Not supported: for, switch, struct, typedef, float, #include, #define, multi-line comments, global initialisers.

Error messages are spartan — usually one line naming the offending token. The compiler bails on first error.

Output

ELF 32-bit LSB relocatable, Tensilica Xtensa.
Loadable on ESP32-S3 via ESP-IDF elf_loader.
