Cellular — TFT emulator

Pixel-faithful preview of what the cellular firmware paints on its ST7735 panel. Two panels supported (isolation/artifacts/cellular/esp32_s3/ for 80×160, esp32_s3_128/ for 128×128) — flip with the panel selector below. Same K=4 hex CA engine ({% static 's3lab/js/engine.mjs' %}), same population GA, same hex-offset tile render. Two render modes: dominant (one colour per population cell) or live (every cell shows a subsampled view of its 16×16 internal CA). The canvas is shown at 4× zoom so every device pixel is a visible 4×4 block.

Algorithm parity

PRNG (xorshift32 + Park-Miller LCG), step_grid, fitness, mutate, palette_inherit, hex neighbour tables — all imported directly from engine.mjs. Same module the live Cellular sublab uses. Tile render mirrors the device: TILE_PX=5, hex stagger via 0.866 × TILE_PX row stride, cell rect sized (TILE_PX-1) × (ROW_STEP-1) for a 1-px black gap on both axes. The C distillation (condenser.distill_cellular.distill_cellular_esp) runs the same algorithm in C.
