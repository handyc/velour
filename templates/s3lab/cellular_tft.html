Algorithm parity

PRNG (xorshift32 + Park-Miller LCG), step_grid, fitness, mutate, palette_inherit, hex neighbour tables — all imported directly from engine.mjs. Same module the live Cellular sublab uses. Tile render mirrors the device: TILE_PX=5, hex stagger via 0.866 × TILE_PX row stride, cell rect sized (TILE_PX-1) × (ROW_STEP-1) for a 1-px black gap on both axes. The C distillation (condenser.distill_cellular.distill_cellular_esp) runs the same algorithm in C.
