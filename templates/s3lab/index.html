      The algorithm here is a line-for-line port of isolation/artifacts/hex_ca_class4/esp32_s3_full/src/main.cpp: same K=4 hex CA, same 14×14 grid, same 4096-byte packed genome, same class-4 fitness, same 4-bit output state-mask, same INPUT_PULLUP input semantics. genome.bin uses the 4104-byte HXC4 ‖ palette ‖ genome tail format from the standalone hunter — files round-trip both directions with no conversion. The hunt runs in a Web Worker so the UI stays responsive.

      Pin map on the SuperMini: TFT uses 4–7 + 11–12; USB D-/D+ is on 19/20; BOOT on 0. Safe defaults for bindings: 1, 2, 3, 8, 9, 10, 13, 14, 21 (~9 free). Typical "logic processor" wiring: 4 inputs + 4 outputs.
